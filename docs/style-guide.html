Minimum Opacity Standard

Readable text should never go below 50 (31%). Axis labels at 50 are the floor. Decorative elements (grid lines, ghost trails) can go as low as 06. Borders should be at least 15 to be perceptible on dark backgrounds.

Current Inconsistencies

Audit of places where similar elements use different values across components.
Chart title font size

CarbonAwareChart & KeplerDashboard use 11px bold. DuckCurveChart, RenewableGrowth, DuckCurveVPP use 13px bold. CurtailmentChart uses 16px bold.

Axis label font size

Y-axis: most use 9px but RenewableGrowthChart uses 10px.
X-axis: DuckCurve/CarbonAware use 9px, Curtailment uses 10px, RenewableGrowth uses 11px.

Legend font size

CarbonAwareChart uses 9px Inter. DuckCurveChart, RenewableGrowth, DuckCurveVPP use 10px Inter.

Grid line opacity suffix

Most charts use textDim + '15' for grid lines. CurtailmentChart uses textDim + '12'. FrequencyLine uses textDim + '20'.

Grid dash pattern

Most charts: [3, 5]. FrequencyLine: [5, 5]. DuckCurveChart base demand: [6, 4]. Negative price zone border: [4, 4].

Bar chart top corner radius

RenewableGrowthChart: [4, 4, 0, 0]. CarbonAwareChart: [3, 3, 0, 0]. CurtailmentChart (annual): [3, 3, 0, 0].

Canvas stat box border radius

DuckCurveChart: 3px. FrequencyDemo: 3px. CurtailmentChart inline cards: 8px.

padLeft across charts

50px (RenewableGrowth, Curtailment), 55px (CarbonAware, DuckCurveVPP), 60px (DuckCurveChart). Without a side panel, 50-55px would standardize well.
