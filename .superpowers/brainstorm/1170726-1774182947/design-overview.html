Page Anatomy (Hybrid: Briefing + Labs)

Page Template
Hero
Title + one-line hook + animated hero visual (e.g., pulsing circuit, spinning turbine)
Briefing Sections (scroll-driven)
2-3 core concepts. Short prose + sticky canvas that animates as you scroll. The "aha moment" delivery system.
Lab Sections (interactive playgrounds)
1-2 standalone simulations. Sliders, drag-and-drop, scenario builders. "Now you try it."
Next Mission Link
"Continue to: [next page]" + optional "Go deeper: [research topic link]"

Design Principles

  • Mission Control, not Classroom — you're a trainee at a grid operations center, not a student in a lecture hall
  • Show, don't tell — every concept gets a visual before text explanation. If it can be interactive, it is.
  • Precise but accessible — real units, real data, real physics. No dumbing down, just clear explanation.
  • Progressive depth — main narrative is scannable in 3 min. Labs reward 15+ min of exploration.
  • Cross-linked — Basics pages link to Learn modules and Research topics for those who want more.
