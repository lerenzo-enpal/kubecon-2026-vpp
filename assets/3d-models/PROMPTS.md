# 3D Model Image Generation Prompts

Reference prompts for ChatGPT image generation. Generate the image, then import into your 3D model generation tool.

---

## Coal Power Plant

Generate a reference image for a 3D model of a coal-fired power plant.

Visual context: This model will be placed as a small 3D icon on a dark satellite-style map in a cinematic presentation. The map uses a near-black background (#0a0e17), with cyan (#22d3ee) glowing UI panels, red (#ef4444) for danger states, amber (#f59e0b) for solar/warning, and green (#10b981) for batteries. The overall aesthetic is "Bloomberg Terminal meets NASA Mission Control" — dark, precise, zero decoration.

Style:
- Medium Dark-toned, non-photorealistic miniature — like a matte architectural maquette lit by cool overhead light in a dark room
- Medium Dark gray / charcoal base materials (not white or bright) — the model must read clearly against a near-black map without looking like a bright blob
- Visibility - while we want it darker, it should slighly stand out on the map so the audience can see it clearly.
- Subtle emissive accents: small glowing cyan lights on equipment, amber glow from furnaces/stacks, red warning indicators — these should be the only bright spots
- Matte, non-reflective surfaces — brushed metal, weathered concrete, composite panels
- Isometric viewing angle (~45 degrees elevation, ~30 degrees rotation)
- Transparent background — the object should float with no background, no ground plane, no shadow plane, no environment. Just the isolated asset ready for compositing onto a dark map
- Minimal geometric complexity — clean forms, no excessive piping or wiring. This needs to work as a small map marker (~40-80px on screen)
- No text, labels, people, vehicles, terrain, or ground plane
- No base plate, pedestal, platform, or floor slab under the object — the structures sit on no surface beneath them. Buildings touch nothing at the bottom
- Subtle ambient occlusion in crevices only
- The feeling should be: a precision-engineered scale model, lit by cool monitor glow, floating in empty space

The asset: Two hyperbolic cooling towers with faint steam wisps, three tall smokestacks with dim red aviation warning lights at the tips, a cluster of industrial buildings with dark roofs. Small cyan indicator lights on control buildings. Overall tone: dark gray concrete and steel.
